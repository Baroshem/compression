This is a Assets Compression module for [Nuxt 3](https://v3.nuxtjs.org).

## Features

- Nuxt 3 ready
- Assets Compression using [vite-plugin-compression](https://github.com/vbenjs/vite-plugin-compression)
- Advanced configuration options
- TypeScript support
